# Heart Disease Data Analysis & Power BI Dashboard

## 📌 Project Overview
This project performs end-to-end data analysis on a heart disease dataset using Python and presents insights through an interactive Power BI dashboard.

## 🛠 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI
- GitHub

## 🔍 Analysis Steps
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Feature grouping and risk analysis
- KPI creation using DAX
- Interactive dashboard design

## 📊 Power BI Dashboard
Includes:
- Total Patients
- Heart Disease Count
- Disease Percentage
- Age Group vs Disease
- Gender Distribution
- Risk factor analysis

## 📂 Project Structure
- `notebook/` – Jupyter analysis
- `data/` – cleaned dataset
- `powerbi/` – Power BI dashboard
- `screenshots/` – dashboard images
